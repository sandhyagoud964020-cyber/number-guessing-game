# number-guessing-game
A beginner-friendly Python Number Guessing Game that uses random number generation, loops, conditions, and input validation. The game provides hints, counts attempts, and allows players to play multiple rounds.
# Number Guessing Game

A simple Python game where the computer randomly selects a number between 1 and 100, and the player tries to guess it.

## Features
- Random number generation
- Hint messages (Too High / Too Low)
- Attempt counter
- Input validation
- Play Again option

## Technologies Used
- Python
- random module

## How to Run

1. Install Python.
2. Open the terminal.
3. Run:

python number_guessing_game.py
